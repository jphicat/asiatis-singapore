# Asiatis Singapore (asiatis.com.sg)

B2B One-Page Landing Site for the Singaporean market.

## Tech Stack
- **HTML5 / CSS3** (Static)
- **Tailwind CSS** (via CDN)
- **Google Fonts** (Outfit)
- **Lucide Icons** (SVG path imports)
- **GitHub Actions** (CI/CD to GitHub Pages)

## Structure
- `index.html`: Main landing page with anchor navigation.
- `assets/`: 
  - `hero-sphere.jpg`: Premium hero background.
  - `logo.png`: Brand logo.
- `favicon.ico`: Site favicon.
- `skills.yaml`: Antigravity IDE skill integration.
- `rules-workflow.json`: B2B inquiry processing rules.

## Navigation Anchors
- `#expertises`: Core B2B service sectors.
- `#languages`: Targeted European & Asian language pairs.
- `#global`: Global presence and experience.
- `#contact`: Business contact card (Email, Phone, Address).

## Deployment
Pushing to the `main` branch triggers the GitHub Actions workflow in `.github/workflows/deploy.yml` to deploy to GitHub Pages.
